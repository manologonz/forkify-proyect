## Forkify Proyect
Forkify is an app where you can search for recepies, and add the ingredients
to your shopping list

## Node version
* Node 12.22.8

## User interface
![Forkify interface](https://drive.google.com/uc?export=view&id=1p1ioPKMp4tGHvWW5lfZ9UAz1POevDLRl)


## Commands
### Setup
```text
npm install
```

## Production

```text
npm build
```

## Development

```text
npm start
```
## Development build

```text
npm run dev
```
